pytools.viz.dendrogram#

Drawer and styles for dendrogram representations of linkage trees.

Classes#

DendrogramDrawer

Draws dendrogram representations of LinkageTree objects.

DendrogramMatplotStyle

Draws dendrograms as trees, using line color and thickness to indicate leaf/branch weights.

DendrogramReportStyle

Renders dendrograms as ASCII graphics for inclusion in plain-text reports.
