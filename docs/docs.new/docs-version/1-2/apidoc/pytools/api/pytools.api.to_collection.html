pytools.api.to_collection¶

pytools.api.to_collection(values, *, element_type=None, arg_name=None)[source]¶

Return the given values as a collection, i.e., an iterable container.

  • if arg values is a collection, return arg values unchanged

  • if arg values is an iterator, return a tuple with its elements

  • if arg values is not an iterable, return a tuple with the value as its only element

Parameters

  • values (Union[Iterable[~T], ~T]) – one or more elements to return as a collection

  • element_type (Union[Type[~T], Tuple[Type, …], None]) – expected type of the values, or a tuple of alternative types of which each value must match at least one

  • arg_name (Optional[str]) – name of the argument as which the values were passed to a function or method; used when composing the TypeError message

Return type

Collection[~T]

Returns

the values as a collection

Raises

TypeError – one or more values did not match the expected type(s)
