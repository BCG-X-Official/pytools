pytools.viz.dendrogram.LinkageTree¶

class pytools.viz.dendrogram.LinkageTree(*, scipy_linkage_matrix, leaf_names, leaf_weights, max_distance=None, distance_label=None, leaf_label=None, weight_label=None)[source]¶

A traversable tree derived from a SciPy linkage matrix.

Supports len(), and numerical indexing and iteration of nodes.

Parameters

  • scipy_linkage_matrix (ndarray) – linkage matrix calculated by function scipy.cluster.hierarchy.linkage()

  • leaf_names (Iterable[str]) – labels of the leaves

  • leaf_weights (Iterable[float]) – weight of the leaves; all values must range between 0.0 and 1.0, and should add up to 1.0

  • max_distance (Optional[float]) – maximum theoretical distance value; this must be equal to, or greater than the maximum distance in arg scipy_linkage_matrix (optional)

  • distance_label (Optional[str]) – a label describing the type and/or unit of distances passed in arg scipy_linkage_matrix (optional)

  • leaf_label (Optional[str]) – a label describing the type of names passed in arg leaf_names (optional)

  • weight_label (Optional[str]) – a label describing the type and/or unit of weights passed in arg leaf_weights (optional)

scipy_linkage_matrix: numpy.ndarray¶

The original linkage matrix created by scipy.cluster.hierarchy.linkage().

One row of the scipy linkage matrix is a quadruple: (<index of left child>, <index of right child>, <distance between children>, <number of descendant nodes>), where the descendant nodes include the nodes from the entire sub-tree, from direct children down to leaf nodes.
