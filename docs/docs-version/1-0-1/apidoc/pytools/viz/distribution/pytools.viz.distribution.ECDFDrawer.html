pytools.viz.distribution.ECDFDrawer¶

class pytools.viz.distribution.ECDFDrawer(style=None, iqr_multiple=1.5, iqr_multiple_far=3.0, hide_far_outliers=False)[source]¶

Drawer for empirical cumulative density functions (ECDFs), highlighting outliers using Tukey’s outlier test.

The drawer highlights samples as outliers or far outliers.

A sample is considered an outlier if it is outside the range \([q_1 - m * \mathit{iqr}, q_3 + m * \mathit{iqr}]\) where \(q_1\) and \(q_3\) are the lower and upper quartiles, \(\mathit{iqr} = q3 - q1\) is the inter-quartile range (IQR), and \(m\) is the IQR multiple.

By convention, common values for \(m\) are \(m = 1.5\) for outliers, and \(m = 3\) for far outliers.
