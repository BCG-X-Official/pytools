pytools.text.format_table¶

pytools.text.format_table(headings, data, formats=None, alignment=None)[source]¶

Print a formatted text table.

Parameters

  • headings (Sequence[str]) – the table headings

  • data (Union[DataFrame, ndarray, Sequence[Sequence[Any]]]) – the table data, as an array-like with shape [n_rows, n_columns]

  • formats (Optional[Sequence[Optional[str]]]) – formatting strings for data in each row (optional); uses str() conversion for any formatting strings stated as None

  • alignment (Optional[Sequence[Optional[str]]]) – text alignment for each column (optional); use "<" to align left, "=" to center, ">" to align right (defaults to left alignment)

Return type

str

Returns

the formatted table as a multi-line string
