pytools.meta.SingletonMeta¶

class pytools.meta.SingletonMeta(*args, **kwargs)[source]¶

Meta-class for singleton classes.

Subsequent instantiations of a singleton class return the identical object. Singleton classes may not accept any parameters upon instantiation.

Definitions

__call__(*args, **kwargs)[source]¶

Return the existing singleton instance, or create a new one if none exists yet.

Behind the scenes, uses a weak reference so the singleton instance can be garbage collected when no longer in use.

Singletons must be instantiated without any parameters.
