pytools.viz.dendrogram.DendrogramHeatmapStyle¶

class pytools.viz.dendrogram.DendrogramHeatmapStyle(*, ax=None, colors=None, min_weight=0.01)[source]¶

Draws dendrograms as a tree composed of color tiles, coloring tiles as a heat map indicating leaf/branch weights.

Bases

DendrogramMatplotStyle

Metaclasses

ABCMeta

Parameters

  • ax (Optional[Axes]) – optional axes object to draw on; create a new figure if not specified

  • colors (Optional[ColorScheme]) – the color scheme to be used by this drawing style

  • min_weight (float) – the minimum weight on the logarithmic feature importance color scale; must be greater than 0 and smaller than 1 (default: 0.01, i.e., 1%)

text_dimensions(text, x=None, y=None, **kwargs)¶

Calculate the horizontal and vertical dimensions of the given text in axis units.

Constructs a Text artist then calculates it size relative to the axis ax managed by this style object. For non-linear axis scales, text size differs depending on placement, so the intended placement (in data coordinates) should be provided.

Parameters

  • text (str) – text to calculate the dimensions for

  • x (Optional[float]) – intended horizontal text placement (optional, defaults to left of view)

  • y (Optional[float]) – intended vertical text placement (optional, defaults to bottom of view)

  • kwargs (Any) – additional keyword arguments to use when constructing the Text artist, e.g., rotation

Return type

Tuple[float, float]

Returns

tuple (width, height) in axis units
