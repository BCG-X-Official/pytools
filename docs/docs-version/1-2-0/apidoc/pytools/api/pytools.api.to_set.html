pytools.api.to_set¶

pytools.api.to_set(values, *, element_type=None, arg_name=None)[source]¶

Return the given values as a set.

  • if arg values is a set, return arg values unchanged

  • if arg values is an iterable other than a set, return a set of its elements

  • if arg values is not an iterable, return a set with the value as its only element

Parameters

  • values (Union[Iterable[~T], ~T]) – one or more elements to return as a set

  • element_type (Union[Type[~T], Tuple[Type, …], None]) – expected type of the values; raise a TypeError if one or more values do not implement this type

  • arg_name (Optional[str]) – name of the argument as which the values were passed to a function or method; used when composing the TypeError message

Return type

Set[~T]

Returns

the values as a set

Raises

TypeError – one or more values did not match the expected type
