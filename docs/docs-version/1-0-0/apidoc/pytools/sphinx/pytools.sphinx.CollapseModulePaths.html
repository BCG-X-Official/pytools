pytools.sphinx.CollapseModulePaths¶

class pytools.sphinx.CollapseModulePaths(collapsible_submodules, collapse_private_modules=True)[source]¶

Replace private module paths with their public prefix so that object references can be matched by intersphinx.

Metaclasses

ABCMeta

Parameters

  • collapsible_submodules (Mapping[str, str]) – mapping from module paths to their public prefix, e.g., {"pandas.core.frame": "pandas"}

  • collapse_private_modules (bool) – if True, collapse module sub-paths consisting of one or more protected modules (i.e. the module name starts with an underscore)
