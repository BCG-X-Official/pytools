pytools.api¶

Basic tools for API development, supporting documentation, deprecation, and run-time validation.

Classes¶

AllTracker

Track global, public items defined in a module and validate that all eligible items have been included in the __all__ variable.

DocValidator

Validates docstrings and type hints in all Python sources in a given directory tree.

Functions¶

deprecated

Decorator to mark a function as deprecated.

deprecation_warning

Issue a deprecation warning.

get_generic_bases

Bugfix version of typing_inspect.get_generic_bases().

inheritdoc

Decorator to inherit docstrings of overridden methods.

is_list_like

Check if the object is list-like.

public_module_prefix

Get the public prefix of the given module name.

subsdoc

Decorator that matches a given pattern in the decorated object’s docstring, and substitutes it with the given replacement string (see re.sub())

to_list

Return the given values as a list.

to_set

Return the given values as a set.

to_tuple

Return the given values as a tuple.

update_forward_references

Replace all forward references with their referenced classes.

validate_element_types

Validate that all elements in the given iterable implement the expected type.

validate_type

Validate that a value implements the expected type.
